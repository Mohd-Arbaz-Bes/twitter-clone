# Twitter Clone

A responsive Twitter clone UI built with Tailwind CSS.  
This project replicates the modern Twitter (X) interface with a clean and responsive design.

## 🚀 Features

- Responsive Twitter UI
- Modern clean design
- Sidebar navigation
- Feed section
- Widgets panel
- Mobile-friendly layout
- Built with reusable components

## 🛠️ Tech Stack
- Tailwind CSS

## 📂 Project Structure

```bash
twitter-clone/
│── public/
│── css/
│   │── input.css
│   │── output.css
│── index.html
│── package-lock.json
│── package.json
│── README.md
```

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/Mohd-Arbaz-Bes/twitter-clone.git
```

2. Go to project folder

```bash
cd twitter-clone
```

3. Install dependencies

```bash
npm install
```

4. Start development server

```bash
npm run dev
```
